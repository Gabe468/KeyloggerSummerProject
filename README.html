Welcome to my Keylogger Project
Goal: To create a simple Key Logger
Project Description:
A keylogger that you can record your keys then save
it to a file to later on replay and open
How to run:
1.Run the main.py file
2.Select an option to record or see recorded files
3.If select 1 then type the keys as you want
4.Save the file with Y and type the name you want for the file
5.Run the program again and press 2 to see list of previously recorded files
6.Type the name of the file you want to play
7.The file plays back
Used Python for coding
Used pynput for key recording and pressing
Used os for directory changes
Used ast to convert string list into list type
